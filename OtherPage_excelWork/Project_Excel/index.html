About Project my work

I created and automated several task to improve work flow and reduce work load for fellow team members. Through the uses of python using imports suck as selenium, openpyxl, pandas, ect. These programs were implemented within other locations to help aswell.

I gather information and create excel tables which other people are able to work out of as everything is shared through google drive. The main objectives that I completed were creating a system to validate revieved information, checking models through a webscapper, and implementing a mass excel data pull to gather laptop specs.

Details about the code

The check system will download all information at a location take in a excel sheet and return and excel sheet with assets that did not pass the checks. Checks basically if statements that throw errors as the information loops through the excel sheet. The webscapper takes in the model that is recorded when team memebers recieve assets. It then goes to the respective support page searches the laptop and gets the support page model. Finally it verify that our model matches the support page model. Lastly laptops are wiped at our location. We use to have people manually putting in the spec such as ram and disk space. I automated the task of doing that by looping through excel sheets that contain the information that get generated during the wiping process.
